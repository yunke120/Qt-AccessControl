## Introduction

​		通过虹软SDK制作的人脸识别门禁系统，支持人脸的检测、识别、注册、删除、查找等，通过`sqlite`数据库进行存储。

## Environment

- QT >= Qt 5
- VS2017 x64
- opencv 2.4.9
